Project Brief
You are rebuilding the complete frontend of SmartAgri — a precision agriculture dashboard — from scratch as a single, self-contained HTML file. This is NOT a mockup. It must work like a real production app in any modern browser. No external CDNs, no React, no frameworks. Pure HTML + CSS + Vanilla JS only.

The final file must open in a browser and immediately feel like a $5M SaaS product. Every element must be intentional. No Lorem Ipsum. No placeholder text. All real data from the app is provided below. The design language is called "Antigravity" — dark, weightless, futuristic precision-farming aesthetic.
